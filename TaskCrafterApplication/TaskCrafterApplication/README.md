# TaskCrafter Desktop Application Skeleton

This is a Java desktop application skeleton for TaskCrafter.

- Uses Java Swing for the UI
- UI style reflects the character of the logo
- Logo is placed in `resources/logo.png` (replace with your actual logo)
- No functionalities implemented yet (see Specifica Funzionale for requirements)
- Ready for build and further development

## Build Instructions

Use Maven to build:

```
mvn clean package
```

## Run Instructions

After building, run:

```
java -cp target/TaskCrafterApplication-1.0-SNAPSHOT.jar taskcrafter.Main
```

## Structure

- `src/` - Java source files
- `resources/` - Logo and other resources
