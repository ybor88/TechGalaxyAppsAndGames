SPECIFICA FUNZIONALE PROGETTO: BusinessFlow ERP (Versione 100% gratuita)

TIPO APPLICAZIONE: Web Application

OBIETTIVO Web application per la gestione della contabilità aziendale e
del ciclo operativo amministrativo. L'applicazione è accessibile via
browser senza installazione locale. Sono incluse esclusivamente
funzionalità basate su software open source o gratuiti, senza API
commerciali, free tier o costi nascosti.

DEPLOY: Avvio locale con un singolo comando. L'applicazione è
immediatamente accessibile nel browser su localhost. Nessuna
configurazione di server, cloud o pipeline richiesta.

STACK TECNOLOGICO

Backend: - Python - FastAPI

Frontend: - Next.js - React - TailwindCSS

Database: - PostgreSQL - pgvector

Servizi: - Redis - Celery - MinIO

AI Locale: - Ollama - Llama 3 / Mistral

OCR: - Tesseract OCR

PDF: - WeasyPrint - ReportLab

MODALITA DI ESECUZIONE

Tipo: Web Application eseguibile in locale, accessibile via browser

Avvio backend:  cd backend && uvicorn app.main:app --reload

Avvio frontend: cd frontend && npm run dev

Accesso: http://localhost:3000 (apre l'app nel browser)

Nessun cloud, nessun server remoto, nessuna pipeline richiesta.
L'intera applicazione gira sulla macchina locale dell'utente.
